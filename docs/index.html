Project Snapshot

Format Benchmarks combines independent benchmark runners and one web dashboard, making results easy to run, compare, and present.

Multi-language stack

Python, Go, Rust, and Java benchmark backends orchestrated by an ASP.NET Core web app.

Metrics that matter

Serialization and deserialization latency, round-trip, memory usage, compression ratio, and throughput.

Built for comparison

Run history, side-by-side run comparison, charted output, plus JSON/CSV export support.
